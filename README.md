# 🛍️ Multi-Platform Poster 多平台商品貼文自動化系統

這是一個為電商商家設計的工具，可自動從商品網址擷取資訊與圖片，產生貼文文案，並一鍵發佈至多個社群平台（Facebook、Instagram、旋轉拍賣）。

---

## 🚀 功能特色

- 🔗 商品網址自動解析（如蝦皮、MOMO）
- 🖼 圖片下載並加上日期標籤
- ✍️ 使用 LLM 自動生成多版本文案（支援選擇語氣）
- 📤 支援自動發文至：
  - Facebook 粉絲專頁
  - Instagram 商業帳號
  - 旋轉拍賣（模擬貼文）
- 📅 支援排程發文與重複貼文
- ✅ 支援多語系、多平台同步
